# Python Quiz Game
A simple, interactive Python quiz game that tests your knowledge of Python programming concepts. 
The quiz reads questions from an external .txt file and gives users a randomized set of questions every session.  
Perfect for beginners learning Python or anyone who wants to challenge themselves.

 ## Features
* Randomly selected questions every playthrough
* Score tracking
* Clean and simple console interface
* Ability to exit the quiz anytime by typing exit or quit
* Reads questions and options from an external question bank
* Validates user input and prevents invalid answers

## Project Structure
quiz_game_project_2/

├── question_bank.txt     # Stores all quiz questions, options, and answers

├── main.py               # Main quiz game logic

└── README.md             # Project documentation

## How It Works
1. You choose how many questions you want for the round.
2. The program randomly selects that many questions from the question bank.
3. For each question, you type a, b, c, or d.
4. Your score is displayed at the end of the section.
5. You can continue to another section or quit the game anytime.

## How to Run the Game
git clone https://github.com/your-username/your-repo-name.git

cd your-repo-name

python main.py
